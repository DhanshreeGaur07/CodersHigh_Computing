# Real-Life Applications of Euclid's GCD Algorithm

Euclid's Greatest Common Divisor (GCD) Algorithm is one of the oldest and most efficient algorithms in mathematics and computer science. Despite its simplicity, it serves as a fundamental building block in many modern technologies and engineering applications.

---

## 1. Cryptography and Cybersecurity

Euclid's Algorithm is extensively used in public-key cryptography, especially in the **RSA encryption algorithm**.

### Applications
- RSA key generation
- Digital signatures
- Secure online transactions
- HTTPS communication
- Authentication systems

The Extended Euclidean Algorithm is used to compute modular inverses, which are essential for encryption and decryption.

## Why This Project Matters

This project not only computes the Greatest Common Divisor but also analyzes the efficiency of Euclid's Algorithm by counting the number of iterations required for different input pairs.

The project experimentally verifies an important theoretical result known as **Lamé's Theorem**, which states that the worst-case running time of Euclid's Algorithm occurs when the input numbers are **consecutive Fibonacci numbers**.
